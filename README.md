# react-ink-snake

# About

This is a small project using react-ink to create a simple snake game clone you can run in your terminal.

## Install

```bash
$ npm install
```


## CLI

```
	Usage
	  $ react-snake

	Options
		--color color_name

	Examples
	  $ react-snake --color=ff00ff
	  $ react-snake --color=red

$ react-snake --help
```
